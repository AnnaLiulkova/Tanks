# Tanks
### Visuals/Audio
* **Середовище:** Налаштовано Post-processing для створення кінематографічної картинки та запечено освітлення (Baked Lighting) для оптимізації продуктивності.
* **Audio System:** Background Music, звуки при натисканні кнопок та покращений звуковий супровід для пострілів.
* **VFX:** Покращено ефекти (Particle System), вибух при знищення танка, а також додано індивідуальні візуальні ефекти для вибухів різних снарядів.

### UI
* **UI Flow:** Головне меню, меню паузи під час бою та екран результатів раунду (зроблені за туторіалом). 

### GameLevels
Реалізовано три ігрові сцени, які гравець може обрати прямо з головного меню:
* **Level Jungle** - стандартна сцена, яка завантажується на початку гри.
* **Level Desert** 
* **Level Moon**
  
### Custom Mechanics
* **Антибонуси:** 
  Механіки штрафів: 
  **SlowDown:** Тимчасове уповільнення танка.
  **ShootingPenalty:** Тимчасове блокування стрільби.
* **Color Swap:** Додано можливість гравцям самостійно обирати та змінювати колір свого танка перед початком бою (4 кольори на вибір).
